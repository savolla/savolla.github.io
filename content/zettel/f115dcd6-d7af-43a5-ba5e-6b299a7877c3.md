-   in `/etc/docker/daemon.json`
    ```json
      {
        "insecure-registries" : ["NEXUS_IP:YOUR_REPO_PORT"]
      }
    ```
-   here is an example
    ```json
      {
        "insecure-registries" : ["192.168.1.54:5000"]
      }
    ```
-   `sudo systemctl restart docker`
