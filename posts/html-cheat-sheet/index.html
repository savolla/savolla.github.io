HTML Cheat Sheet

how to

  • create a heading?

      <h1>Heading</h1>

  • create a paragraph element?

      <p>paragraph</p>

  • create an image element? notice there are no closing forward slashes in img elements

      <img src="path/to/image">

  • break a line?

      <br/>

  • tell html to use latest html version? by putting the docstring

      <!DOCTYPE html>

  • generate all initial html structure with emmet? ! + TAB

  • generate a div with a class with emmet? div.class_name + TAB

  • generate a div with an id with emmet? div#id_name + TAB

  • create 5 div elements with unique class names? .container$*5

  • get dummy text for experimenting? NUMBER is a number of words you want lorem__NUMBER__ + TAB

  • get 5 dummy texts that has 4 words per line ? lorem__WORDS__*__LINES__

  • display an image ?

      <img src="./path/to/image.png" alt="alternative text"/>

  • display images from internet (external images)?

      <img src="http://www.addrsofpic.com/pic.png" alt="blah" />

  • adjust the size of very big images? just use width. height will be adjusted automatically as soon as we don’t specify it explicitly.

      <img src="./path/to/img.png" width="256" alt="pexels" />

  • add links?

      <a href="www.google.com">GOOGLE</a>

  • add link that opens a new tab when opened? use target="_blank"

      <a href="www.google.com" target="_blank">GOOGLE</a>

  • add internal link? (in site link) specify html file path in link element

      <a href="./path/to/html">Home</a>

  • scroll and focus on an element with a link?

      <a id="top" href="#bottom">go bottom</a>
  <!-- Some looooooong text (5000 lines) -->
  <a id="bottom href="#top">go to top</a>

  • convert an image into a link? use a>img + TAB

      <a href="www.google.com">
    <img src="google_icon.png" alt="icon"/>
  </a>

  • use special characters in html? like copyright?

      <p>copyright &copy;</p>

      <p>left &larr;</p>
  <p>right &rarr;</p>
  <p>up &uarr;</p>
  <p>down &darr;</p>

      <ul>

    <li>michael</li>

  • what is the difference between radio buttons and check box ?

    • radio buttons: choose only one
    • check boxes: choose multiple
