CSS Cheat Sheet

how to

  1. name a css file? style.css or main.css

  2. how to link style.css with index.html ? in html file,

       <link href="./style.css" rel="stylesheet"/>

  3. write internal css? put your css code inside style tags

          <head>
        <style>
          h3 {
            color : red;
          }
        </style>
      </head>

  4. how comments look in css?

       /* this is a comment */

  5. change h1,h2 and h3 at the same selector?

       h1,h2,h3 {
       color : olive;
   }

  6. change background color of the entire page?

       body {
       background-color : black;
   }

  7. style by classes or id?

       <h3 class="cls0"> TITLE0 </h3>
   <h3 id="cls1"> TITLE1 </h3>

       .cls0 {
       color : red;
   }

   #cls1 {
       color : blue;
   }

  8. set space between two lines?

       line-height : 1.5em;

  9. set font ?

       font-family : monospace;

  10. set a background image?

         background-image: url("../img/bg.png");

  11. set opacity of text? use rgba() with a value between 1 and 0 at the end

         .class0 {
         color : rgba(100,53,81,0.5);
     }

  12. make links without underline ?

         a {
         text-decoration : none;
     }

  13. cancel all the default margins?

            *{
            margin : 0;
        }

  14. change font of an element?

         .element {
         font-family : Fira Code;
     }

  15. use google fonts

    1. go to google fonts page
    2. select your font
    3. click select this style
    4. copy part and paste is between element
    5. copy CSS rules and paste it to CSS file where you want to use that font
    6. click on @import section and copy the code inside <style> element
    7. paste it into the top of your CSS file

  16. make a font bolder? use font-weight

         .element {
         font-weight: bold;
     }

  17. make font italic? use font-style

         .element {
         font-style : italic;
     }

  18. reach a nested element inside a class? use > syntax

         <div class="cls0">
       <ul>
         <li>item</li>
         <li>item</li>
       </ul>
     </div>

         cls0 > ul > li {
         /* your code */
     }

  19. increase space between letters in a paragraph?

         .element {
         letter-spacing: 5px;
     }

  20. increase space between words ?

         .element {
         word-spacing: 5px;
     }

  21. capitalize an entire paragraph ?

         .element {
         text-transform: capitalize;
     }

  22. add hover action to an element?

             .element:hover {
         /* code */
         }

  23. add opacity to an element?

         .element {
         opacity: 0.5;
     }

  24. add a border around an element?

         .element {
         border-style: solid;
     }

  25. add border width of an element?

         .element {
         border-width: 10px;
     }

  26. overlap two elements? with negative margin

  27. make list items on one line?

         <ul>
         <li>item1</li>
         <li>item2</li>
         <li>item3</li>
     </ul>

         ul > li {
        display: inline;
     }

  28. display text in the center of the element? use text-align: center

  29. create a navigation bar of a website?

         <div class="navbar">
       <a href="#">Home</a>
       <a href="#">Contact</a>
       <a href="#">Coffee</a>
       <a href="#">Music</a>
     </div>

         .navbar a {
         font-family: Fira Code;
         margin-right: 20px;
         font-size: 1rem;
         text-decoration: none;
     }

     .navbar {
         background: black;
     }

  30. add a custom font to the css? add the following on the top of css file

         @font-face {
         font-family: "font-name";
         src: url("../fonts/my-font.ttf");
     }

  31. set opacity of an element? use opacity attribute

         .element {
         opacity: 0.5;
     }

  32. use image as a background instead of straight color?

         .element {
         background: url("/path/to/image.extension");
     }

  33. make an image non-repeatable?

         .element {
         background: url("/path/to/image.extension");
         background-repeat: no-repeat;
     }

  34. spread the image into one area (div) ?

         .element {
         background: url("/path/to/image.extension");
         background-size: cover;
     }

  35. center the background image?

         .element {
         background-position: center;
     }

  36. adjust the position by x and y values of a background image? use persentages to adjust

         .element {
         background: url("/path/to/image.extension");
         background-position: 20% 50%;
     }

  37. add an image with protected aspect ratio?

         .element {
         background: url("/path/to/image.jpg");
         background-size: contain;
     }

  38. align text to the center?

         .element {
         text-align: center;
     }

  39. make a parallax effect? use background-attachment

         .banner {
         background: url("../img/bg.jpg");
         background-attachment: fixed;
     }

  40. make a gradient element? use grey and white

         .element {
         background-attachment: linear-gradient(grey, white);
     }

  41. give angle to the gradient?

         .element {
         background: linear-gradient(45deg, white, black);
     }

  42. make a nice transition effect ? when user scrolls down the transparent screen comes on the background image. the background image does not change btw

         <div class="first">
     </div>

     <div class="second">
     </div>

         .first {
         background: url("../img/bg2.jpg");
         background-attachment: fixed;
         background-size: cover;
         background-position: center;
         background-repeat: no-repeat;
         height: 100vh;
     }

     .second {
         background: url("../img/bg2.jpg");
         background-attachment: fixed;
         background-size: cover;
         background-position: center;
         background-repeat: no-repeat;
         opacity: 0.5;
         height: 100vh;
     }

what is

  • what css stands for? cascading style sheets

  • what is the syntax for css?

      selector{
      property0 : value0;
      property1 : value1;
  }

  • what is the difference between span and div

    • div is usefull for grouping elements together
    • span is usefull for assigning some class or id names inline
    • when elements grouped with div it will cause a new line
    • hovewer span won’t cause a new line

  • what is the best way to represent a color ? using rgb

      h1 {
      color : rgb(100,100,100);
  }

  • what is the most primitive way to represent size? pixels.

      .cls0 {
      font-size : 12px;
  }

  • what is the drawbacks of representing something in pixels? since pixels are constant and depends on nothing, it makes the element non-responsive. so the size will not change when screen size change

  • what is font stack ? it is a concept where we present different font families to the web browser. if web browser fails to render the first font family then it tryes out the second one. if web browser fails at every font then the last one (monospace) font will be assumed

    example usage;

      .element {
      font-family : 'Courier New' Courier, monospace;
  }

  • what is css box mode ?

      +-------------------------------------+
  |               Margin                |
  |  +-------------------------------+  |
  |  |            Border             |  |
  |  |  +-------------------------+  |  |
  |  |  |         Padding         |  |  |
  |  |  |  +-------------------+  |  |  |
  |  |  |  |      Content      |  |  |  |
  |  |  |  +-------------------+  |  |  |
  |  |  +-------------------------+  |  |
  |  +-------------------------------+  |
  +-------------------------------------+

  • what is the shortcut way of writin borders? parameter order matters!

      .element {
      /* width style and color */
      border: 5px solid blue;
  }

  • what is the difference between border and outline ?

    • borders are like glued to the content. we can’t put space between content and border

    • outlines in other hand they are more flexible than borders. also we can use only outlines instead of borders. we can give offset to outlines which borders can’t quite handle.

             .element {
           outline: 0.2rem solid black;
           outline-offset: 10px /* set outline 10px above the content */
           outline-offset: -10px /* set outline 10px inside the content */
       }

  • what are gradients ? it’s a color effect. two or more colors are mixed into one element. and result will look like an oval object. used in button coloring

  • what happens when we use width and height in persentage? the persentage will work depending on the parent div’s sizes. for example if parent div has 20px height and we give 50% height to the child then child will have 10px height

      .parent {
      height : 20px;
      width : 20px;
  }
  .child {
      height : 50%; /* 20%50 = 10 */
      width : 50%; /* 20%50 = 10 */
  }

  • persentages will depend on … parent values!

  • what vw and vh do? vw and vh are measure units in css. they are relative values depend on screen size. when browser window size change, then those values are also change example;

      <div class="cls0">
  </div>
  <div class="cls1">
  </div>

      .cls0 {
      background-color : orange;
      width : 50vw;
      height : 50vh;
  }

  .cls1 {
      background-color : yellow;
      width : 50vw;
      height : 50vh;
      margin-left: 50vw;
  }

  • what calc() function does in css? it takes two values with px, vh, vw, rem, em etc and applies arithmatic operations on them.

  • use case scenario of calc() function assume we have a navbar on top of the page and it has 100px height. now let’s say we need a banner below navbar. this banner will have height of entire page (without scroll bar). how do we adjust that height automatically?

    in this case we can use calc() function like this:

      <div class="navbar">
  </div>

  <div class="banner">
  </div>

      *{
      margin : 0; /* zero out default margins */
  }

  .navbar {
      background : blue;
      height : 100px;
  }

  .banner {
      background : green;
      height : calc(100vh - 100px); /* 100vh - navbar.height */
  }
