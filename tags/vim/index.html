permanent macros in vim

01 Jan 0001, 00:00
vim / howto / tools

this is a short step by step tutorial to save your vim macro and use it everytime fire up vim do q a create your macro press q to finish exit vim :q! vim ~/.vimrc type let @q = '' put your cursor on first single quote do " a p macro should be pasted inside single quotes like: let @q = 'your_macro_content' save and exit ESC :wq fire up vim again do @ q now you should have your macro saved.
